Terjadi perpindahan secara osmosis, di mana zat pada larutan A akan berpindah ke larutan B.

Penjelasan:

Osmosis adalah peristiwa perpindahan zat secara pasif di mana zat akan berpindah dari larutan kurang pekat (konsentrasi pelarut rendah/encer) ke larutan ke lebih pekat (konsentrasi pelarut tinggi) melalui membran semipermeabel. Semipermeabel berarti hanya zat-zat tertentu yang dapat menembus membran ini.

Pada gambar dapat dilihat bahwa konsentrasi zat (titik hitam) lebih sedikit pada bagian A dan bagian B memiliki lebih banyak zat. Sementara itu, terdapat membran semipermeabel (garis putus-putus) di antara kedua larutan tersebut. Perbedaan konsentrasi kedua larutan akan menyebabkan perpindahan zat dari larutan A ke larutan B.

9. Soal fisika SMA ttg relativitas, mohon dijawab ya? :), no. 31

• penjumlahan Kecepatan

Va = v
Vb = -v
Vab = 1,5 v
v = __?

Vab = (Va - Vb) / (1 - Va Vb / c²)
1,5 v = (v - (-v)) / (1 - v • (-v) / c²)
1,5 v = 2v / (1 + v² / c²)
1,5 v = 2v / (1 + (v/c)²)
1 + (v/c)² = 2v / 1,5v
1 + (v/c)² = 4/3
(v/c)² = ⅓
v/c = ⅓√3

Bagaimana cara menginstall aplikasi pada laptop/komputer?

Jika telah memiliki file instalasi aplikasinya,

1. Buka File Explorer

2. Buka folder dimana file instalasi aplikasi berada (file umunya berekstensi .exe atau .msi)

3. Klik 2x (Double klik) pada file tersebut

4. Baca dan ikuti instruksi pada dialog instalasi yang muncul (misalnya setujui EULA, klik tombol Next, dll.)

5. Tunggu proses intalasi selesai, dan klik tombol Finish

6. Aplikasi telah terinstall

• Penjelas:

Untuk membuka aplikasi yang telah terinstall, dapat klik 2x pada shortcut aplikasi yang ada di Dekstop, atau dengan Klik tombol start > All program > Klik aplikasi yang telah diinstall.

Silahkan download pada aplikasi yang memungkinkan pada sistemnya, seperti untuk android maka menggunakan google play store, dll. Atau bisa mencarinya dan mendownloadnya pada link yang memungkinkan di google atau yang lainnyamapel : ti

bagaimana cara memasang aplikasi di laptop?

jawab :
kamu dapat memasang aplikasi di laptop melalui :
1. store
2. web resmi aplikasi

bagaimana caranya? cara ini ditunjukkan untuk instal aplikasi dari internet
1. buka web aplikasinya
2. ketika tersedia tombol "download" klik tombolnya
3. ketika ada pop up berupa file aplikasinya dari browser, klik download
4. ketika sudah, klik filenya dan ketika ada pop up UAC (Windows), klik allow
5. ketika sudah muncul jendela tentang penginstalan aplikasi (sambutan) klik next
6. ketika sudah muncul "EULA", baca dulu. ketika setuju, klik next
7. tentukan tempat dimana kamu menginstal aplikasinya. biasanya ditandai dengan drive C: atau D:
8. pilih fitur yang ingin diinstal jika fitur tersebut tidak digunakan, jangan diinstal karena akan memakan penyimpanan
9. ketika sudah, klik next. ketika ada bar loading, penginstalan sedang berjalan. tunggu hingga selesai
10. ketika sudah selesai, klik finish. ketika ada pilihan "run this program after finish the installation", klik saja jika kamu ingin menjalankan program secara langsung

perlu diketahui bahwa ada banyak file instalasi berisi virus. diharapkan untuk mengunduh dan menginstal program diwebsite resminya. dan juga program juga harus sesuai dengan spesifikasi komputer dan arsitekturnya. jika salah / tidak diperhatikan akan mengakibatkan program gagal terinstal dan gagal beroperasi. contoh : programnya x64 tapi komputermu x86 / 32 bit. programnya ga bisa diinstal maupun dijalankan.

38. Sebuah laptop memiliki tenaga cukup untuk menyala tanpa aplikasi selama 6 jam. Di dalamnya ada sebuah aplikasi yang jika dinyalakan membuat tenaga laptop hanya bertahan selama 4 jam. Jika dari tenaga penuh laptop itu menyala dengan aplikasi selama 40 menit sebelum kemudian aplikasi itu dimatikan, berapa sisa waktu agar tenaga laptop itu habis?


Jawaban:

300 menit/5 jam

Penjelasan:

Misal tenaga laptop x

Diketahui

Kecepatan tenaga laptop habis tanpa aplikasi x/6 jam atau x/360 menit

Kecepatan tenaga laptop habis dengan aplikasi x/4 jam atau x/240 menit

Laptop dipakai dengan aplikasi selama 40 menit maka sisa tenaga

x - x/240 * 40 = x - x/6

Lalu digunakan tanpa apliaksi maka waktu sampai habis

t = sisa tenaga / kecepatan habis

t = x-x/6 / x/360

t = x- x/6 * 360/x

t = 360-60=300 menit
