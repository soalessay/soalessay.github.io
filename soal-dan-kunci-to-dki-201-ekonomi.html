6. 5,9,7,…201 tentukan banyak suku jika suku terakhir adalah 201


Barisan Aritmatika

Runus suku ke-n

Un = a + (n - 1)b

a = U1 = suku pertama

b = beda = U2 - U1

soal diperbaiki

5, 7, 9, ... , 201

a = 5

b = 7 - 5 = 2

Un = 201

a + (n - 1)b = 201

5 + 2(n - 1) = 201

5 + 2n - 2 = 201

2n + 3 = 201

2n = 201 - 3

2n = 198

n = 198/2

n = 99

Banyak suku (n) jika suku terakhirnya 201,

adalah 99 suku.
