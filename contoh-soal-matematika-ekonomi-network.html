6. Buatlah contoh pertanyaan network planning dan permasalahannya!


Apa saja manfaat network planning?

- Perencanaan suatu proyek yang kompleks.  

- Scheduling pekerjaan-pekerjaan sedemikian rupa dalam urutan yang praktis dan efisien.  

- Mengadakan pembagian kerja dari tenaga kerja dan dana yang tersedia.  

- Scheduling ulang untuk mengatasi hambatan-hambatan dan keterlambatan-keterlambatan.

- Menentukan Trade Off (kemungkinan pertukaran) antara waktu dan biaya.  

- Menentukan probabilitas penyelesaian suatu proyek tertentu.

15. Jelaskan apa yang dimaksud dengan Network id dan host id dan berikan contohnya


Pada dasarnya Network ID sendiri dapat kita artikan sebagai suatu Identitas dari suatu kelompok IP Address atau subnet, yang mana Network ID ialah suatu bagian dari IP address yang menunjukan letak network atau jaringan dari suatu IP Address. Sedangkan untuk Host ID sendiri ialah suatu nomor perangkat keras fisik Ethernet Network Interface Card atau yang sering disingkat dengan sebutan NIC.

Sebagai salah satu contohnya ialah:

Kelas A yaitu 10.11.12.1 maka untuk contoh ini network ID nya ialah 10. sedangkan untuk Host ID ialah 11.12.1.Kelas B yaitu 188.10.11.1 maka untuk contoh ini network ID nya ialah 188.10. sedangkan untuk Host ID ialah 11.1.Kelas C yaitu 192.168.1.1 maka untuk contoh ini network ID nya ialah 192.168.1 sedangkan untuk Host ID ialah 1.

Pembahasan

Pada umumnya Net ID sendiri dapat berperan penting dalam mengidentifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network, yang mana pada konteks ini dapat kita simpul kan bahwa seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama.

Diantara 110 mahasiswa, 37 orang mempelajari matematika, 25 orang mempelajari data mining, 45  orang mempelajari computer network, 15 mempelajari matematika dan computer network, 7  mempelajari matematika dan data mining, 10 mempelajari data mining dan computer network dan 30  tidak mempelajari satupun diantara ketiga bidang tersebut. Maka tentukan :

a) Diagram Venn (JAWABAN DI LAMPIRAN)

b) Hitunglah banyaknya mahasiswa yang mempelajari ketiga bidang tersebut. (5 orang)

DIKETAHUI

Semesta (Total Mahasiswa) = 110 mahasiswa

A (matematika) = 37 mahasiswa

B (data-mining) = 25 mahasiswa

C (computer-network) = 45 mahasiswa

A ∩ C (matematika dan computer-network) = 15 mahasiswa

A ∩ B (matematika dan data-mining) = 7 mahasiswa

B ∩ C (data-mining dan computer-network) = 10 mahasiswa

A' (Mahasiswa yang tidak mempelajari dari 3 pelajaran) = 30 mahasiswa

.

DITANYA

A. Diagram Venn (watermark digunakan agar tidak ada tindak copas)

B. Hitunglah banyaknya mahasiswa yang mempelajari ketiga bidang tersebut.3

.

PENYELESAIAN

A) Check Lampiran yaa

B) Hitung terlebih dahulu yang mempelajari satu bidang saja

A saja (Matematika saja)

= (37 - 15 - 7)

= 15

B saja (data-mining saja)

= (25 - 7 - 10)

= 8

C saja (computer-network saja)

= (45 - 15 - 10)

= 20

Sekarang mencari nilai yang mempelajari ketiga bidang

(110 - 30) = 15 + 20 + 8 + 15 + 7 + 10

80 = 75

80 - 75 = x

5 = x

x = 5.

Jadi yang mempelajari ketiga bidang adalah 5 orang

Jawaban:

Storage Area Network (SAN) merupakan solusi konfigurasi masa depan dalam media penyimpanan data dalam jumlah besar (TeraByte) dalam berbagai servis yang berbasis online di Internet maupun IntraNet.

Sebagai contoh, apabila host kehabisan disk space, maka SAN dapat mengalokasikan storage lebih banyak pada host tersebut. Biasanya tersambung melalui Fiber Channel, sebuah teknologi komunikasi data berkecepatan sangat tinggi, menjadikan SAN sebuah jaringan dedicated yang platform-independent yang beroperasi dibelakang server.

36. contoh aplikasi osi dari network layer

Berfungsi untuk mendefinisikan alamat-alamat IP, membuat Header untuk paket-paket dan kemudian melakukan routing melalui internet-working dengan menggunakan router dan switch layer 3.
- Protocol pada layer ini yaitu : Routing
- Perangkat Network layer : Router
