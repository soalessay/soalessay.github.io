5. cari pengertian/istilah pemrograman dasar web/script script HTML?


HTML merupakan bahasa dasar pembuatan web. Disebut dasar karena dalam membuat web, jika hanya menggunakan HTML, tampilan web kita akan terasa hambar. Terdapat banyak bahasa pemograman web yang ditujukan untuk memanipulasi kode HTML, seperti JavaScript dan PHP. Namun sebelum anda belajar JavaScript maupun PHP, memahami HTML merupakan hal yang paling awal. HTML bukan bahasa pemograman (programming language), tetapi bahasa markup (markup language), hal ini terdengar sedikit aneh, tapi jika anda telah mengenal bahasa pemograman lain, dalam HTML tidak akan ditemukan struktur yang biasa di temukan dalam bahasa pemograman seperti IF, LOOP, maupun variabel. HTML hanya sebuah bahasa struktur yang fungsinya untuk menandai bagian-bagian dari sebuah halaman. Selain HTML, dikenal juga xHTML yang merupakan singkatan dari eXtensible Hypertext Markup Language. xHTML merupakan versi tambahan dari HTML. Untuk tahap ini anda boleh menyebut bahwa xHTML itu juga HTML

9. Mengapa HTML bukan termasuk bahasa pemrograman?


Jawaban:

HTML adalah singkatan dari Hypertext Markup Language. Pengertian HTML sering kali dikaitkan dengan bahasa pemrograman. Akan tetapi, sejatinya, HTML bukan sebuah bahasa pemrograman (programming languange), melainkan sebuah markup languange.

Jadi HTML yang merupakan bahasa markup tidak termasuk bahasa pemrograman karena tidak memiliki tag atau kode yang berfungi untuk menyelesaikan masalah. HTML hanya berfungsi memberikan tampilan luar bagi bahasa pemrograman web seperti php dan JavaScript, bersama dengan CSS.

1. Fungsi JavaScript Dalam Pemograman Web

JavaScript pada awal perkembangannya berfungsi untuk membuat interaksi antara user dengan situs web menjadi lebih cepat tanpa harus menunggu pemrosesan di web server. Sebelumjavascript, setiap interaksi dari user harus diproses oleh web server.

Bayangkan ketika kita mengisi form registrasi untuk pendaftaran sebuah situs web, lalu men-klik tombol submit, menunggu sekitar 20 detik untuk website memproses isian form tersebut, dan mendapati halaman yang menyatakan bahwa terdapat kolom form yang masih belum diisi.

2. JavaScript Engine adalah mekanisme internal yang dimiliki oleh web browser.JavaScript Engine dapat diumpamakan dengan compiler dalam bahasa pemograman lain, yakni algoritma yang digunakan untuk menjalankan JavaScript. Semakin cepat sebuah web browser menjalankan JavaScript akan semakin baik.

Biasanya disetiap rilis baru web browser seperti Google Chrome, Internet Explorer, maupun Mozilla Firefox, juga diikuti rilis terbaru JavaScript Engine yang menawarkan kecepatan lebih baik.

22. Carilah informasi tentang pemrograman web​


Jawaban: Pemrograman Web

Penjelasan:

adalah proses atau cara untuk menjalankan intruks Pada sebuah komputer yang terhubung ke internet untuk membuat fungsi atau tugas tertentu

Jawaban dan Penjelasan:

Pemrograman web, atau web programming, adalah aktivitas menulis kode-kode program dalam proses atau tahapan pengembangan/pembangunan web (web development), yang di antaranya meliputi konten web, scripting web dari sisi client dan server, serta keamanan jaringan, terkait dengan web yang dibangun.

Pemrograman web berbeda dengan hanya “memprogram” atau “membuat program”, karena memerlukan pengetahuan interdisipliner (antara beberapa disiplin ilmu yang berhubungan) pada area aplikasi, scripting di sisi client dan server, serta teknologi basisdata.

Oleh karena itu, secara garis besar, terdapat dua buah kategorisasi pemrograman web, yaitu:

Pemrograman web pada sisi client
Pada sisi client, pemrograman diperlukan dalam hal pengaksesan data dan penyajian informasi dari dan kepada pengguna web.
Pemrograman web pada sisi client juga diperlukan untuk meningkatkan pengalaman pengguna (user experience) dalam berinteraksi dengan web yang dikunjungi.
Untuk hal ini, seorang web programmer setidaknya harus menguasai pengolahan DOM (Document Object Model), HTML, XML, Javascript, dan CSS. Pemrograman web pada sisi server
Pada sisi server, pemrograman terutama diperlukan dalam hal pengambilan data (dari database), keamanan jaringan dan website, serta kinerja web.
Bahasa pemrograman yang dapat digunakan adalah PHP, ASP, Java, Python, dan lain-lain.
Untuk database, yang umum digunakan adalah MySQL atau MariaDb, namun tidak jarang juga yang menggunakan MS SQL Server dan PostgreSQL, atau bahkan file teks biasa yang terorganisir dan sistematis.
