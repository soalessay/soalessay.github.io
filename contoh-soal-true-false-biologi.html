8. Buatlah fungsi untuk mendeteksi bilangan primer pada deret kemudian menghasilkan deret bilangan Boolean sesuai dengan input: Contoh: input = [17,18,12,11,10] Ouput:[true, false, false, true, false]


Jawaban:

def is_prime(n):

   """Fungsi untuk mendeteksi apakah bilangan n prima"""

   if n <= 1:

       return False

   for i in range(2, int(n**0.5) + 1):

       if n % i == 0:

           return False

   return True

def prime_boolean(arr):

   """Fungsi untuk menghasilkan deret bilangan Boolean sesuai dengan input"""

   boolean_arr = []

   for num in arr:

       boolean_arr.append(is_prime(num))

   return boolean_arr

# Contoh penggunaan

input_arr = [17, 18, 12, 11, 10]

output_arr = prime_boolean(input_arr)

print(output_arr) # Output: [True, False, False, True, False]

Fungsi is_prime digunakan untuk mendeteksi apakah suatu bilangan merupakan bilangan prima atau bukan. Fungsi ini menggunakan metode brute-force untuk memeriksa faktor-faktor bilangan tersebut.

Fungsi prime_boolean digunakan untuk menghasilkan deret bilangan Boolean sesuai dengan input. Fungsi ini memanggil fungsi is_prime untuk setiap bilangan dalam input, dan menghasilkan deret bilangan Boolean dengan nilai True jika bilangan tersebut prima, dan False jika tidak.

33. apa perbedaan antara TRUE dan FALSE​


Jawaban:

True= benar

False=salah

Penjelasan:

Semoga membantu :)

Jawaban:

perbedaan antara nilai TRUE dan FALSE adalah

True diidentifikasikan sebagai 1 sedangkan False diidentifikasikan sebagai 0

Dalam sisi hardware, nilai true direpresentasikan dengan kondisi logika HIGH, nilai false direpresentasikan dengan logika LOW

Pada elektronika digital, kondisi True dapat direpresentasikan dengan tegangan tinggi sesuai vcc, sedangkan kondisi False dapat direpresentasikan dengan tegangan rendah atau ground.

Pembahasan

nilai True dan False adalah sebuah nilai boolean, yaitu nilai yang hanya berisi 2 kondisi, benar dan salah, 1 atau 0, HIGH dan LOW.

Penggunaan nilai True dan False sangat penting dalam pemrograman, elektronika, dan hal hal lainnya yang berkaitan dengan sistem pengambilan keputusan. Semua kondisi dalam proses yang bercabang akan menghasilkan nilai True jika kondisi terpenuhi, dan nilai False ketika kondisi tidak terpenuhi.

Dari nilai true dan false yang sederhana inilah yang memungkinkan adanya teknologi canggih dalam pengambilan keputusan seperti teknologi AI (Artificial Intelegence).

Selain itu, nilai true false juga sering direpresentasikan dengan nilai biner. Bilangan biner memiliki 2 nilai saja yaitu 1 dan 0. Pada sistem komputer, nilai true dan false dibedakan dari nilai biner yang merepresentasikannya. 0 pada biner dianggap False, dan 1 pada biner dianggap True.
