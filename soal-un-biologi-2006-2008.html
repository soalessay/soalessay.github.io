10. Diketahui segitiga MAB dengan AB = 300cm sudut MAB = 60derajat dan sudut ABM = 75derajat, Maka AM adalah ? Soal UN SMA 2008


AM / sin ABM = AB / sin AMB
AMB = 180° - 60° - 75° = 45° 
AM / sin 75° = 300 / sin 45° 
AM / sin (45 + 30) = 300 / (1/2 √2)
AM / (sin 45 . cos 30 + cos 45 . sin 30) = 600 / √2
AM / (1/2 √2 . 1/2 √3 + 1/2 √2 . 1/2 ) = 300√2
AM / (√6 + √2) / 4 = 300√2
4AM / (√6 + √2) = 300√2
4AM = 300√2(√6 + √2)
4AM = 300√12 + 600
4AM = 600√3 + 600
AM = 150√3 + 150 cm

20. nilai dari 2009^2 - 2008^2 + 2007^2 - 2006^2 + 2005^2 adalah


Jawaban:

nilai dari 2009^2 - 2008^2 + 2007^2 - 2006^2 + 2005^2 adalah 17,005.

Penjelasan dengan langkah-langkah:

Kita bisa menghitung nilai dari setiap kuadrat terlebih dahulu, kemudian menjumlahkannya secara bergantian.

2009^2 = 4,036,081

2008^2 = 4,032,064

2007^2 = 4,028,049

2006^2 = 4,024,036

2005^2 = 4,020,025

Maka,

2009^2 - 2008^2 + 2007^2 - 2006^2 + 2005^2

= (4,036,081) - (4,032,064) + (4,028,049) - (4,024,036) + (4,020,025)

= 17,005

Jadi, nilai dari 2009^2 - 2008^2 + 2007^2 - 2006^2 + 2005^2 adalah 17,005.

Misalkan x = 2007,

maka:

2009 = 2007 + 2 = x + 2

2008 = 2007 + 1 = x + 1

2007 = x

2006 = 2007 - 1 = x - 1

2005 = 2007 - 2 = x - 2

Jadi,

2009^2 - 2008^2 + 2007^2 - 2006^2 + 2005^2

= (x + 2)^2 - (x + 1)^2 + x^2 - (x - 1)^2 + (x - 2)^2

= (x + 2)^2 + (x - 2)^2 - ((x + 1)^2 + (x - 1))^2 + x^2

= (x^2 + 4x + 4) + (x^2 - 4x + 4) - ((x^2 + 2x + 1) + (x^2 - 2x + 1)) + x^2

= (2x^2 + 8) - (2x^2 + 2) + x^2

= x^2 + 6

= 2007^2 + 6

= 4028055

38. nilai dari 2007³-2006(2007)2008/2007² =

2006 = a - 1
2008 = a + 1

(2007³ - 2006.2007.2008) / 2007²
= (a³ - (a - 1)(a)(a + 1)) / a²
= (a³ - a(a² - 1²)) / a²
= (a³ - a³ + a) / a²
= a / a²
