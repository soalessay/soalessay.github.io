4. Pada soal ini, kamu diminta untuk membuat soal dengan pattern/pola IF CLAUSE. Oleh karena itu, kamu harus mengerti terlebih dahulu pola-pola If Clause.


Jawaban:

sophisticated aircraft be built before your eyes at the Boeing factory in Mukilteo.

Penjelasan:

1. If Clause + Reminder

Merupakan bentuk kalimat yang digunakan untuk mengingatkan konsekuensi yang harus dilakukan jika menginginkan sesuatu.

IF + SIMPLE PRESENT, S + HAVE TO/HAS TO/ MUST + V1

Contoh:

Dad: Do you want to join the next mathematics competition?

Tom: Yes, I do dad, and I really want to be the winner.

Dad: If you want to be the winner, you have to study hard.

Tom: Thanks, dad.

2. If Clause + Suggestion

If clause ini digunakan didalam kalimat untuk memberikan suatu saran mengenai suatu tindakan yang harus dilakukan jika menginginkan suatu kejadian terjadi. Bentuk saran dan bentuk peringatan berbeda, saran bisa dilakukan atau tidak (tergantung level), sedangkan peringatan harus dilakukan karena hal itu merupakan suatu konsekuensi.

IF + SIMPLE PRESENT, S + SHOULD /HAD BETTER/ OUGH TO + V1

Contoh:

Dad: Why do you look so pale?

Tom: Hmm I’m sick dad. I may get a fever.

Dad: Ok, If you want to get better soon, you should get some medicine and take a rest.

Tom: Thanks, dad.

IF + SIMPLE PRESENT, WHY DON’T + S + V1?

Jennie: I really like watching, but sometimes I don’t know the latest movie. Do you have any idea?

Anna: If you want to know the latest movie, why don’t you search for it on the internet?

Jennie: That’s a good idea. Thanks

3. If Clause + Imperative

Digunakan untuk menyuruh seseorang untuk melakukan sesuatu jika meginginkan sebuah kondisi terjadi.

IF + SIMPLE PRESENT, V1 + O

Contoh: If you want to know her more, follow her on Instagram.

﻿

4. If Clause + General Truth

Kalimat ini digunakan untuk menyatakan fakta yang terjadi atau akan terjadi jika menginginkan sebuah kondisi terjadi.

IF + SIMPLE PRESENT, SIMPLE FUTURE/PRESENT

Contoh: If you are 17 years old or older, you are allowed to have an id card.

﻿

5. If Clause + Dream

Digunakan untuk menyatakan sebuah mimpi di masa depan tentang apa saja yang mungkin terjadi.

IF + SIMPLE PRESENT, SIMPLE FUTURE

Contoh: If I become a doctor, I will be a good doctor.

9. rumus if Clause dan main Clause dengan baik dan benar​


Jawaban:

Conditional

Tipe I:

1. If S (I, you, we, they) + V1 (simple present tense), S + will + V1 (future tense)

2. If S (he, she, it) + V1 + s/ es (simple present tense), S + will + V1 (future tense)

3. S + will + V1 (future tense) if S (I, you, we, they) + V1 (simple present tense)

4. S + will + v1 (future tense) if S (he, she, it) + V1 + s/ es (simple present tense)

Tipe II:

1. If S + V2 (simple past tense), S + would + V1 (past future tense)

2. S + would + V1 (past future tense) if S + V2 (simple past tense)

Kalau to be, semua pakai were walaupun dia subjeknya I, he, she, it

Tipe III:

1. If S + had + V3 (past perfect tense), S + would have + V3 (future past perfect tense)

2. S + would have + V3 (future past perfect tense) if S + had + V3 (past perfect tense)

• If you marry me, I will make you happy.

• If I am chosen, I will dedicate all my life.

24. contoh if clause + an imperative dan if clause to show a dream ?


1. IF + PRESENT SIMPLE, ...+ IMPERATIVE

Menggunakan form imperative untuk memberikan instruksi atau saran:

If you see John, tell him he needs to come to my office.If you travel abroad, remember your passportIf you go to Australia, send me a postcard.If you go to the supermarket, bring back some milk.If Mike comes, call me.

Kata yang dicetak tebal merupakan imperative atau kata perintah. Kalimat-kalimat tersebut sama saja dengan mengucapkan: Jika kondisi pertama terjadi, kamu harus melakukan hal ini (kondisi kedua/kata perintah).

Pembahasan Conditional Sentences Type 1, 2, 3, and 0

1. First Conditional

First Conditional digunakan untuk pengandaian ke masa depan yang belum dan mungkin terjadi.

POLA kalimat: If + Present Tense, will + infinitive

Contoh:  

If you visit your grandma, she will give you a delicious cake. If my mother knows this, she will be angry.

2. Second Conditional

Second Conditional digunakan untuk pengandaian ke masa lalu yang tidak mungkin terjadi di masa sekarang. Dengan kata lain, sesuatu dapat terjadi saat ini jika sesuatu yang lain di masa lalu dilakukan.

POLA kalimat: If + Past Tense, would/could + infinitive

Contoh:

If we took U turn, we would not get lost. If I were a president, I would give high salary for teachers.

3. Third Conditional

Third Conditional digunakan untuk pengandaian ke masa lalu yang tidak mungkin karena sudah tidak terjadi (di masa lalu).

POLA kalimat: If + Past Perfect Tense, would/could have + past participle

Contoh:

If you had warned me, I wouldn't have gotten an accident. If she had come to the competition, she could have won.

Catatan Penting!!!

Untuk Second Conditional, 'was' selalu digantikan 'were' untuk semua subjek. Contoh: If he were not your brother, I would not lend my motorbike to him.

4. Zero Conditional

Zero conditional digunakan untuk pengandaian yang menyatakan kenyataan yang akan terjadi secara alami, dapat dijelaskan dengan alasan ilmiah. Akibat dari pengandaian ini selalu benar.

POLA kalimat: If + Present Tense, Present Tense

Contoh:

If I drink coffee, I feel queasy. If the flowers don't get enough water, they die.

PADA SOAL

If clause reminder untuk mengingatkan sesuatu akan terjadi jika sesuatu yang lain dilakukan. If clause ini menggunakan CS (conditional sentence) Type 1.

If clause imperative, saat terjadi sesuatu, maka seseorang memberi perintah untuk melakukan sesuatu yang lain. Setelah if clause, kata kerja kalimat kedua adalah bentuk perintah.

If clause suggestion, jika terjadi sesuatu maka sesuatu yang lain kemungkinan akan terjadi, sebagai saran. If clause ini diikuti kata 'should' apda kalimat utamanya.

If clause general truth, jika sesuatu terjadi maka sesuatu yang lain pasti terjadi sebagai konsekuensinya. If clause ini menggunakan CS Type 0.

If clause to show a dream, jika terjadi sesuatu maka sesuatu yang lain dapat dicapai atau dapat dilakukan. If clause ini dapat menggunakan if clause Type 2.
