16. Perbedaan mixed conditional dan If conditional


║MIXED CONDITIONAL VS IF CONDITIONAL║

[tex]Explanation[/tex]

Mixed Conditional dikatakan sebagai mixed time . Baik mixed conditional maupun if conditional, kedua nya sama-sama menjabarkan dampak/konsekuensi dalam suatu kondisi tertentu.

Dalam Mixed conditional, terdapat perbedaan struktur tenses (waktu ) yang ditempatkan pada –if clause –nya dengan tenses yang ada di main clause -nya (result clause). Ini merupakan hasil campuran  tenses ( mixed time ) yang diterapkan dari seluruh type pada  if conditional.  

→ If conditional terbagi dalam 3 kategori utama  terstruktur :

1. Conditional 1 => untuk menyatakan keinginan yang mungkin akan terwujud segera apabila syaratnya sudah terpenuhi saat ini { Real present conditional  }  

Pattern=> If+S+verb 1(s/es)+C , S+will+verb 1 + C

2. Conditional 2 => untuk menyatakan keinginan yang tidak mungkin  terwujud karena persyaratannya tidak  mampu untuk dipenuhi saat ini juga atau nanti-nanti { Unreal present conditional }

Pattern=>If+S+verb 2 + C , S+ would +verb 1 + C  

3. Conditional 3 => untuk menyatakan keinginan yang tidak terwujud di waktu lampau karena persyaratannya tidak terpenuhi { Unreal past conditional }  

Pattern=> If+S+had+verb 3+C , S+would +have+verb 3 + C

Sedangkan, dalam Mixed conditional  merupakan hasil campuran dari semua  kategori type –if conditional yang telah disebutkan di atas. Untuk mengetahui perbedaan mixed conditional dan –if conditional lebih jelas, dengan mencermati tenses yang digunakan dalam kondisi fakta nya.  

ANALYSIS DATA

1. Fact  : The students use Brainly application now  because they got the best mark yesterday

Conditional : If the students hadn’t gotten the best mark yesterday, they wouldn’t use Brainly application now.

Result ↑↓ :

Fakta pertama disimpulkan bahwa terdapat campuran tenses dalam makna kondisi faktanya.  

Maka,  verb “ hadn’t gotten “ merupakan penjabaran “got ” dan ini menggunakan  struktur –if clause dalam –If conditional tipe ke 3 untuk menyatakan kejadian lampau (past) tidak nyata. Sedangkan     “wouldn’t use” merupakan  penjabaran “ use”  dan ini menggunakan struktur –result clause  dalam –if conditional tipe ke 2 untuk menyatakan kejadian terkait  sekarang  (present  ) tidak nyata .

2. Fact  : The students got the best mark because they used Brainly application yesterday.

Conditional : If the students hadn’t gotten the best mark, they  wouldn’t have used Brainly application yesterday.

Result ↑↓ :

Fakta kedua disimpulkan bahwa tidak terdapat campuran tenses karena ini termasuk kategori terstruktur dari  type 3 conditional  untuk menyatakan kondisi fakta di waktu lampau (past).

“hadn’t gotten” merupakan penjabaran “got” dan “wouldn’t have used” merupakan penjabaran “used.” Bentuk kata kerja dalam If clause dan main clause sama-sama menyatakan kejadian terkait waktu lampau { past} tidak nyata,  jika dilihat tenses dari makna kalimat kondisi faktanya. Maka, ini termasuk kalimat type 3 conditional

Dari hasil analysis, terlihat jelas perbedaan antara mixed conditional dan –if conditional . Pada mixed conditional menggunakan pola tenses yang berbeda (campuran) dalam makna fakta nya dan terdapat kolaborasi  struktur kata kerja pada –if clause nya maupun main clause –nya yang diterapkan dari masing-masing  tipe conditional baik tipe 3 maupun tipe 2

Sedangkan –if conditional menggunakan pola tenses yang sama dalam makna faktanya dan terstruktur berdasarkan pola masing-masing tipe –if conditional.

19. Fach dari soal conditional sentence If he cassed me i would ran a way


║CONDITIONAL SENTENCE║

[tex]Answer[/tex]

If he chased me, I would ran away.

Fact => I won't run away because he can't chase me.

[tex]Discussion[/tex]

Pernyataan diatas termasuk kalimat conditional tipe ke 2, dimana menjelaskan bahwa terdapat kekontrasan apa yang diucapkan dengan kondisi kenyataannya saat ini. Kalimat conditional tipe ke 2 memaparkan bahwa si pembicara memiliki niatan untuk melakukan sesuatu tetapi tidak didukung dengan situasi yang ada pada saat ini ataupun mendatang.

[tex]Explanation[/tex]

Conditional sentence structure consists -If clause and result clause ( main clause ).

→There are 4 types of Conditional Sentence←

1. ZERO CONDITIONAL

The Zero Conditional is used to describe the Phenomenon, and General truth ( Fact  ). Zero Conditional uses the simple present tense form to reflect the truth.

Pattern :

If+S+verb (s/es)+O , S+verb (s/es)+O

If+simple present, simple present

Example :

If we boil water , it evaporates up into atmosphere.

{ Jika kita merebus air, itu menguap ke udara }

Analysis :

→ It tells about the general truth or phenomenon.

→ It uses the simple present tense form to describe the truth. It is proven that the water evaporates up and disappear when it is heated at 100 degrees Celsius.

2. TYPE 1 CONDITIONAL

The Type 1 Conditional is used to describe the truth  in the present/future ( The planning (prediction)  will be probably happened ). In fact, the speaker will actualize his/her dream now or soon .

Pattern :

If +S+ verb (s/es) + O, S+will+verb1+O/C

If + simple present, simple future

Example :

If I study hard, I will pass the exam easily.

{ Jika saya belajar giat, saya akan lulus ujian dengan mudah }

Analysis

→ It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true. Meaning : it's possible that I will be successful for the exam later.

3. TYPE 2 CONDITIONAL

The Type 2 Conditional type  is used to describe  the untruth in the present / future. It is Contrary to the fact in the present or it could be in the future. The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future

"Keep in mind, if we use the first , second person and third person singular  ( I, she, he, it , ) in the nominal sentence , it must be followed by the PAST linking verb "were."" It is the same way to the plural form as the formal writing." If we use the PAST linking verb "was " , it should be used in the colloquial as the informal conversation.

Pattern:

If+S+verb2+O/C , S+would+verb1+O/C

If + simple past , past future

Example :

If I were a president, I would build the largest library in the Southeast Asia.

{ Andai saya seorang presiden, saya akan bangun perpustakaan terbesar se Asia Tenggara }

Analysis :

→ It uses the simple past form to reflect the untruth condition in the present and the speaker's intention will not come true because it is contrary to the real condition for now or maybe future. Meaning :  I can't build the largest library because I'm not a president now.

4. TYPE 3 CONDITIONAL

The Type 3 Conditional  Is used to describe  the untruth in the past. It is contrary to the fact in the past and It  reflects a regret for something that the speaker couldn't afford to do in the past.

Pattern :

If+S+Had+Verb 3+O  , S+Would+have+Verb 3+O

If + past perfect , past future perfect

Example :

If I had known my friend's house address correctly, I wouldn't have been lost.

{ Jika saya tahu alamat rumah teman saya dengan jelas, saya tidak akan telah kesasar } 

Meaning :  I didn't know my friend's house address so I was lost.

PEMBAHASAN:

Conditional Sentence

if conditional sentence merupakansebuah kalimat pengandaian yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi.

Ada tiga jenis kalimat if conditional:

1. If Conditional Type 1

If conditional type 1 yang juga disebut Future conditional sentence adalah sebuah kalimat pengandaian yang belum terjadi dan kemungkinan besar akan terjadi dimasa mendatang (future) jika syaratnya (anak kalimatnya) terpenuhi.

Rumus/Pola kalimat:

If + S + V1 (simple present), S + will + Verb 1

Contoh:  

If I go to Palembang I will meet Sarah  If we have a yacht, we will sail the seven seas

2. If Conditional Type 2

If conditional type 2 yang juga disebut present conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak mungkin terpenuhi, dan apa yang diandaikan bertolak belakang dengan kenyataan yang ada.

Rumus/ Pola kalimat:

If + S + V2 (simple past), S + would + Verb 1

Contoh:  

If I were you, I would forgive him If we had a yacht, we would sail the seven seas

3. If Conditional Type 3

If conditional type 3 yang juga disebut past conditional sentence adalah sebuah kalimat pengandaian yang digunakan untuk mengandaikan sesuatu yang tidak akan pernah terjadi karena syarat sub clausenya tidak akan pernah terpenuhi, karena apa yang diandaikan sudah berlalu di masa lampau (masanya sudah terlewati).

Rumus/ Pola kalimat:

If + S + had + V3 ( past perfect), S + would have + Verb3

Contoh :  

If Mimi had done her homework, she would have not got a punishment. If we had had a yacht, we would have sailed the seven seas

Kalimat passive (passive voice)

Passive voice merupakan suatu bentuk kalimat dimana subjek kalimat menerima aksi, bukan melakukan aksi. Tidak seperti active voice yang fokus terhadap pihak yang melakukan aksi (doer of action), bentuk ini lebih berfokus kepada pihak atau objek yang menerima hasil dari suatu aksi tersebut (receiver of action).

Rumus Kalimat Pasif ( Passive Voice)

Subject(object dari kalimat aktif) + to be ( disesuaikan dengan bentuk tensesnya + Verb 3 (past participle)

TENSES                                              RUMUS PASSIVE VOICE

Simple present                                subject + is/am/are + Verb 3

Present continuous                     subject + is/am/are + being + Verb 3

Present perfect                              subject + have/has been + Verb 3

Simple past                                     subject + was/were + Verb 3

Past continuous                         subject + was/were + being + Verb 3

Past perfect                                     subject + had been + Verb 3

Simple future                                   subject + will be + Verb 3

Be going to                              subject + is/am/are + going to + be + Verb 3

Future perfect                                subject + will have been + Verb 3

Modal                                              Subject + Modal + be + Verb 3

Langkah-langkah (tips) membentuk kalimat pasif (Passive Voice) dari kalimat aktif:

1. Pilih objek penderita dari kalimat aktif itu

2. Buatlah objek tersebut menjadi subjek dari kalimat pasifnya

3. Perhatiakan bentuk tensesnya, subjeknya tunggal atau jamak karena tenses yang di gunakan harus sama

4. bentuklah predikatnya dengan to be + Verb 3

37. Pertanyaan bonus :Jelaskan tentang if conditional!


If conditional Showing or used to show that something is true or happens only if something else true or happens.
Dari pengertian diatas bisa diambil kesimpulan bahwa if conditional sentence adalah sebuah kalimat yang digunakan untuk menunjukkan bahwa sesuatu benar-benar terjadi/nyata jika syarat yang lain terpenuhi. Kenapa harus begitu? Karena dalam pola kalimat if conditional sentence terdiri dari 2 klausa, yaitu main clause (kalimat utama) dan yang kedua sub clause (anak kalimat). Jadi kalimat If conditional sentence itu bisa benar-benar terjadi dan mungkin juga tidak akan pernah terjadi. Dan keduanya bisa saling bertukar tempat (dibolak balik). 40. Explain about Conditional If​


Penjelasan:

CONDITIONAL IF

 Conditional if artinya jika / andai, digunakan untuk memperkirakan / memprediksi tentang apa yang dapat terjadi, yang mungkin terjadi dan yang diharapkan akan terjadi.

a) Types the differences among types 1, 2, and 3

 Type 1 : Pengandaian ini mungkin terjadi jika syaratnya terpenuhi. Pada main

clause menggunakan present future dan pada if clause menggunakan

simple present. Type 1 digunakan pada situasi nyata di masa kini atau di

masa depan (future).

• Pattern : * Present future + if + simple present

* If + simple present, present future (will)

• Example : I will come if you invite me

Main clause If clause

 Type 2 : Kalimat yang tidak terjadi di masa lampau.

• Pattern : * Past future + if + past tense

* If + past tense, past future

• Example : I would come if you invited me

Main clause (past future) If clause (past tense)

 Type 3 : Pengandaian yang tidak mungkin terjadi, baik di masa depan maupun

masa lampau.

• Pattern : * Future perfect + if + past perfect

* If + past perfect, future perfect

• Example : I you had invited me I would have come

If clause (past perfect) Main clause (past future perfect)

` 5 examples of type 1 :

1. If we meet at 09.30. So, we will have plenty of time.

2. She will be happier if she has more friends.

3. The zoo keeper will punish her, if she feeds the animals.

4. If it rains, she will take a taxi.

5. I will marry someone famous if I am the movie star.

` 5 examples of type 2 :

1. If we met at 09.30. So, we would have plenty of time.

2. She would be happier, if she had more friends.

3. The zoo keeper would punish her, if she fed the animals.

4. If it rained, she would take a taxi.

5. I would marry someone famous, if I were the movie star.

` 5 examples of type 3 :

1. If we had met at 09.30. We would have had plenty of time.
