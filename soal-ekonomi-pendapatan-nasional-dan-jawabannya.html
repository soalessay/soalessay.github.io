Persamaan agregat demand (AD) dapat dituliskan sebagai berikut:

AD = C + I + G + (X-M)

= 0,75Y + I + 45 + (X-5-1%Y)

Persamaan agregat supply (AS) dapat dituliskan sebagai berikut:

AS = Y = C + S + T

= 0,75Y + S + (Y-T)

= 0,75Y + 5%Y + 45 + (X-5-1%Y)

= 1,75Y + 45 + (X-5-1%Y)

Y = 45 + (X-5-1%Y) / 0,25

Y = 180 + (X-5-1%Y)

C = 0,75Y

(c) Saving (S):

S = Y - C - T

= Y - 0,75Y - (Y-T)

= 0,25Y - (Y-T)

= 0,25Y + T - Y

= T - 0,75Y
