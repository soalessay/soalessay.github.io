Pembahasan

Setelah kalian mempelajari materi pada bagian B ini, coba rumuskan sebuah kesimpulan tentang arti penting dari pokok pikiran Pembukaan Undang-Undang Dasar Negara Republik Indonesia Tahun 1945.

Kesimpulannya adalah pokok pikiran yang terkandung dalam Pembukaan Undang-Undang Dasar Negara Republik Indonesia Tahun 1945 adalah sumber hukum tertinggi di Indonesia. Sehingga arti penting sebagai konsekuensi dari kedudukannya adalah seluruh peraturan perundang-undangan di Indonesia harus bersumber pada Pembukaan Undang-Undang Dasar Negara Republik Indonesia Tahun 1945 yang didalamnya terkandung asas kerohanian negara yaitu Pancasila.

Selain sebagai sumber hukum tertinggi, arti penting Pembukaan Undang-Undang Dasar Negara Republik Indonesia Tahun 1945, juga memiliki arti penting dalam konteks hukum dasar tidka tertulis yang disebut dengan konvensi.

15. Kunci jawaban paket mtk halaman 69 kls 9 semester 1 kurikulum 2013


Jawabannya adalah yang ada di pembahasan ya!

Pembahasan

Halo adik-adik! Balik lagi di Brainly!! Gimana, masih semangat belajar kah? Nah untuk pertanyaan di atas itu sedikit masuk ke materi tentang fungsi atau persamaan kuadrat. Yuukk langsung aja kakak jelasin materi kali ini, oke? Oke! Pertama-tama apa itu fungsi kuadrat? Kakak jawab yaaa! Suatu persamaan yang memiliki variabel dimana variabel tersebut mempunyai pangkat yang paling tinggi dua maka persamaan tersebut disebut sebagai fungsi kuadrat. Bentuk umum dari fungsi kuadrat adalah [tex]y = f(x) = ax^2 + bx + c[/tex], dengan keterangan x disebut sebagai variabel, kemudian a, b adalah koefisien dan c adalah suatu konstanta, serta nilai dari a tidak sama dengan 0. Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Persamaan kuadrat : [tex]x^2 + 5x + 6 = 0[/tex]

Untuk p = 1, jika q = 6 maka pq = 6, diperoleh p + q = 1 + 6 = 7

Untuk p = 2, jika q = 3 maka pq = 6, diperoleh p + q = 2 + 3 = 5

Untuk p = 3, jika q = 2 maka pq = 6, diperoleh p + q = 3 + 2 = 5

Untuk p = 6, jika q = 1 maka pq = 6, diperoleh p + q = 6 + 1 = 7

Untuk p = -1, jika q = -6 maka pq = 6, diperoleh p + q = -1 + (-6) = -1 - 6 = -7

Untuk p = -2, jika q = -3 maka pq = 6, diperoleh p + q = -2 + (-3) = -2 - 3 = -5

Untuk p = -3, jika q = -2 maka pq = 6, diperoleh p + q = -3 + (-2) = -3 - 2 = -5

Untuk p = -6, jika q = -1 maka pq = 6, diperoleh p + q = -6 + (-1) = -6 - 1 = -7

Karena harus memenuhi pq = 6 kemudian karena juga harus memenuhi p + q = 5, maka berdasarkan tabel tersebut pada baris kedua didapat p = 2 dan q = 3 atau berdasarkan baris ketiga ditulis p = 3 dan q = 2 (dua hasil ini merupakan hasil yang sama). Sehingga didapat pemfaktorannya

[tex]x^2 + 5x + 6 = (x + 2)(x + 3)[/tex]

Dengan demikian akar-akarnya adalah x = -2 dan x = -3

Persamaan kuadrat : [tex]x^2 + x - 6 = 0[/tex]

Untuk p = 1, jika q = -6 maka pq = -6, diperoleh p + q = 1 + (-6) = 1 - 6 = -5

Untuk p = 2, jika q = -3 maka pq = -6, diperoleh p + q = 2 + (-3) = 2 - 3 = -1

Untuk p = 3, jika q = -2 maka pq = -6, diperoleh p + q = 3 + (-2) = 3 - 2 = 1

Untuk p = 6, jika q = -1 maka pq = -6, diperoleh p + q = 6 + (-1) = 6 - 1 = 5

Untuk p = -1, jika q = 6 maka pq = -6, diperoleh p + q = -1 + 6 = 5

Untuk p = -2, jika q = 3 maka pq = -6, diperoleh p + q = -2 + 3 = 1

Untuk p = -3, jika q = 2 maka pq = -6, diperoleh p + q = -3 + 2 = -1

Untuk p = -6, jika q = 1 maka pq = -6, diperoleh p + q = -6 + 1 = -5

Karena harus memenuhi pq = -6 kemudian karena juga harus memenuhi p + q = 1, maka berdasarkan tabel tersebut pada baris keenam ditulis p = -2 dan q = 3 (dua hasil ini merupakan hasil yang sama). Sehingga didapat pemfaktorannya

[tex]x^2 + x - 6 = (x + 3)(x - 2)[/tex]

Dengan demikian akar-akarnya adalah x = -3 dan x = 2

Persamaan kuadrat : [tex]x^2 - x - 6 = 0[/tex]

Untuk p = 1, jika q = -6 maka pq = -6, diperoleh p + q = 1 + (-6) = 1 - 6 = -5

Untuk p = 2, jika q = -3 maka pq = -6, diperoleh p + q = 2 + (-3) = 2 - 3 = -1

Untuk p = 3, jika q = -2 maka pq = -6, diperoleh p + q = 3 + (-2) = 3 - 2 = 1

Untuk p = 6, jika q = -1 maka pq = -6, diperoleh p + q = 6 + (-1) = 6 - 1 = 5

Untuk p = -1, jika q = 6 maka pq = -6, diperoleh p + q = -1 + 6 = 5

Untuk p = -2, jika q = 3 maka pq = -6, diperoleh p + q = -2 + 3 = 1

Untuk p = -3, jika q = 2 maka pq = -6, diperoleh p + q = -3 + 2 = -1

Untuk p = -6, jika q = 1 maka pq = -6, diperoleh p + q = -6 + 1 = -5

Karena harus memenuhi pq = -6 kemudian karena juga harus memenuhi p + q = -1, maka berdasarkan tabel tersebut pada baris kedua didapat p = 2 dan q = -3 atau berdasarkan baris ketujuh ditulis p = -3 dan q = 2 (dua hasil ini merupakan hasil yang sama). Sehingga didapat pemfaktorannya

[tex]x^2 - x - 6 = (x + 2)(x - 3)[/tex]

Dengan demikian akar-akarnya adalah x = -2 dan x = 3

26. jawaban lks bahasa inggris kls 8 halaman 7 semester 2 kurikulum 2013


Daily Routines

Daily routines atau Daily activities adalah kegiatan yang dilakukan sehari-hari. Kegiatan daily routines dalam Bahasa Indonesia sering juga disebut sebagai kebiasaan/ habit.

Saat kita ingin menyatakan kegiatan sehari-hari/ daily routines, kita perlu menggunakan Simple Present Tense. Simple Present Tense adalah tenses yang digunakan untuk menyatakan kejadian yang berlangsung sehari-hari (Daily routines).

Kata keterangan yang sering digunakan dalam simple present tense adalah:

• Every day

• Every week

• every month

• every year

• once a week

• twice a day

• three times a month

• three times a year

• always

• often

• usually

• seldom

• rarely

• dan lain lain

Pattern yang digunakan dalam simple present tense adalah:

(+) S + V1 atau S + V1s/es

(-) S + do/ does + V1

(?) Do/ DOes + S + V1?

NB:

V1 dan Do untuk subject I, You, They, We

V1s/es dan Does untuk subject He, She, It

Pembahasan

Daily Activities pada teks di atas:

The writer usually wakes up at 5.00 in the morning.He always brushes his teeth, and shaves, and washes his face after alarm wakes up him.He always gets dressed and goes downstair to have breakfast.He always has coffee, cereal and lots of fruits.He always goes work by bus and arrives at his office at 8.00.He finishes his work at 5.00 in the evening and goes home.He and his family always eat dinner in the dining room at around 6.00.They really enjoy watching their favourite TV programs together two or three times a week.He does exercise in the evening.At 9.30, he always goes to bed, reads for a while, and then falls a sleep.

Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33
= (3 x 3 x 3 x 3 x 3 x 3)

= (3 x 3 x 3)
(6 - 3) faktor
= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor
= 4 x 4 x 4 x 4 x 4 x 4
(3 x 2) faktor
= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (32)3 = 33 x 23.
Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)
= (3 x 3 x 3) x (2 x 2 x 2)
3 faktor 3 faktor
= 33 x 23

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.
Contoh soal:
Dengan cara menuliskan dalam bentuk pembagian, tunjukkan bahwa (5 : 3)4 = 54 : 34.
Jawab:
(5 : 3)4 = (5/3)4
= (5/3) x (5/3) x (5/3) X (5/3)
