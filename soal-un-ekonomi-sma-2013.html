Lim x → ∞  (√(ax² + bx + c) - √(px²  + qx + r))
= (b - q)/(2√a) ----->> (Jika a = p)
= ∞ ------------------->> (Jika a > p)
= -∞ ------------------>> (Jika a < p)

Penyelesaian:
Lim x → ∞     (3x - 1 - √(9x²  + 3x - 5))
= Lim x → ∞  (√(3x - 1)² - √(9x²  + 3x - 5))
= Lim x → ∞  (√(9x² - 6x + 1) - √(9x²  + 3x - 5))
= (b - q)/(2√a)
= (-6 - 3)/(2√9)
= -9/(2.3)
= -9/6
= -3/2

5. Latihan Untuk UN besok. Diambil dari UN SMA / SMK 2013 Kode soal 28 No 25. Yang menjawab tanpa langkah langkah / cara akan langsung dihapus oleh Moderator ya! Diketahui segi-8 beraturan dengan panjang jari jari lingkaran luar r cm. Keliling segi 8 tersebut adalah


ambil salahsatu segitiga samakaki 
sudut pusat = 360/8 = 45° 
panjang sisi merupakan panjang tali busur (anggap a) 
tali busur a² = r² + r² - 2*r*r cos 45 
a² = 2r² - 2r² cos 45 
= 2r²(1 – cos 45) 
= 2r²(1 – ½√2) 
a = √(2r²(1 – ½√2)) 
= √(2r²((2 – √2)/2) 
= √(r²(2 – √2)) 
= r√(2 – √2)

Jadi Jawabannya A

8. soal UN 2013/2014 no.13


maaf harus di poto :)
clik gambar biar liat penuh :)Diket dan dit:
*n NaOH = 100 mL x 0,008 M = 0.8 mmol
*n CH3COOH = 100 mL x 0,008 M = 0,8 mmol
*Ka = 10^-5
*Ksp Fe(OH)2 = 1,6 x 10^-15
*[Fe2+] = ? M

A. Mencari nilai [OH-]
--> Soal UN ini mengaitkan 2 bab (kyk SBMPTN, ya), yaitu hidrolisis garam dan hasil kali kelarutan (Ksp), sehingga:
     CH3COOH + NaOH --> CH3COONa + H2O
m        0.8           0,8
b        -0.8          -0.8              +0.8          +0.8
s           -              -                 0.8

[G] = 0.8 mmol / 200 mL = 4 x 10^-3

Bersisa garamnya sebesar 0,8 mmol, masukkan ke rumus hidrolisis:
[OH-] = √Kw x [G]/Ka
         = √10^-14 x 4 x 10^-3 / 10^-5
         = 2 x 10^-6

B. Mencari [Fe2+] dari Fe(OH)2
--> Untuk mencarinya, buat reaksi ionisasi Besi (II) hidroksida tsb:
Fe(OH)2 ⇄ Fe2+ + 2OH-

Ksp = [Fe2+] [OH-]^2
1,6 x 10^-15 = [Fe2+] [2 x 10^-6]^2
[Fe2+] = 1,6 x 10^-15 / 4 x 10^-12
           = 0.4 x 10^-3
           = 4 x 10^-4 Molar (D)
