27. 0,2-0,4 berapa jawanbannya sekalian jalannya


Math

= - 0,2

#HelpingTogether1). 0,2 - 0,4, hasilnya berapa ?

jawaban :

➡ semua angka mau pecahan mau desimal mau bilangan biasa kalau dikurang dengan angka yang lebih besar akan lihat menghasilkan bilangan Minus.

✅ 0,2 - 0,4 =

➡ dan kalau ingin menggunakan angka dalam desimal kita kurangkan angka yang kedua dengan angka pertama akan ditemukan hasilnya tetapi dalam bentuk minus.

✔ 0,2 - 0,4 sama saja dengan 0,4 - 0,2 tetapi hasilnya minus.

✅ 0,4 - 0,2 = 0,2 ( hasilnya akan minus )

✔ 0,2 = - 0,2

➡ Jadi hasil dari 0,2 - 0,4 = - 0,2

= 20+5 = 25°C (d)

seri = -1

jumlah seri = 3×(-1) = -3

Paris = -3°C ke 17°C = 17+3=20
Mexico = 17°C ke 34°C = 34-17=17
Tokyo = -2°C ke 25°C = 25+2=27
perubahan suhu terbesar di Tokyo (d)

suhu permukaan laut = 33°C
jika ketinggian 1750m maka,
1750:250=7
suhu permukaan laut ketika ketinggiannya 1750m
= 7×(-3)°C = -21°C
= 33 + (-21)°C = 12°C (d)

6. -7-(-5) = -7+5 = -2 (b)
7. [4+(-9)-(-3)] - [-5+7+(-8)] = [-5-(-3)] - [2+(-8)] = (-5+3) - (-6) = -2+6 = 4
