15. contoh pengulangaan ( looping ) pada organisasi


Jawaban:

Perulangan (Lopping)

1. Perulangan while

While merupakan instruksi perulangan yang mirip dengan perulangan for. Bentuk perulangan while dikendalikan oleh syarat tertentu , yaitu perulangan akan terus dilaksanakan selama syarat tersebut terpenuhi.

Bentuk umum pernyataan While adalah:

while(syarat)

{

Pernyataan/perintah;

}

Berikut contoh program sederhana menampilkan bilangan ganjil antar 1-10 dengan menggunakan While:

//Program Sederhana Dengan While

#include <stdio.h>

#include <conio.h>

#include <iostream.h>

main()

{

int nilai=1;

clrscr();

while(nilai<=10)

{

cout<<nilai<<" ";

nilai+=2;

}

getch();

}

Hasil outputnya nanti seperti ini

1 3 5 7 9

2. Pernyataan Do....While...

- Pada perulangan Do-While, tubuh loop berupa pernyataan.

- Pada pernyataan Do, mula-mula pernyataan dijalankan.

- Selanjutnya, kondisi diuji. Seandainya kondisi bernilai benar, maka pernyataan dijalankan lagi. Kemudian kondisi diperiksa kembali dan seterusnya.

- Kalau kondisii bernilai salah, maka pernyataan tidak dijalankan lagi

- Pada dasarnya struktur perulangan Do...While sama saja dengan struktur while, hanya saja pada proses perulangan dengan while, seleksi berada di while yang letaknya berada di atas sementara pada perulangan Do...While seleksi berada dibawah batas perulangan. Jadi dengan menggunakan Do...While sekurang-kurangnya akan terjadi satu kali perulangan.

Bentuk umum Do-While

do

{

Perintah/Pernyataan;

}

While(syarat);

Berikut contoh program sederhana menampilkan bilangan ganjil antara 1-10 dengan Do While :

//Program Sederhana Dengan Perintah Do While

#include <stdio.h>

#include <conio.h>

#include <iostream.h>

main()

{

int nilai=1;

clrscr();

do

{

cout<<nilai<<" ";

nilai+=2;

}

while(nilai<=10);

getch();

}

Diketahui:

tc = 40oC

Ditanyakan: tR, tF, dan T

Jawab:

a. Mengubah skala celcius ke reamur

perbandingan skala termometer reamur dan celcius adalah sebagai berikut.

tR

4

tC

tR = 4/5 × tC

tF – 32

31. nilai rata rata ulangan bahasa indonesia kelas 9a 10 orang 8,0. jika nilai rata rata ulanga dodi dimasukkan maka nila rata rata berubah menjadi 7,8. berapakah ulangan dodi?


10 orang (jumlah siswa 9a) x 8,0 = 80
11 orang (siswa 9a+Dodi  ) x 7,8 = 85,8

jadi nilai Dodi = 85,8-80
                    = 5,8 

rata'' 1 = ∑x / n
   8,0  = ∑x / 10
8 x 10 = ∑x
  80     = ∑x

rata'' 2 = ∑x + x / n+1
  7,8   = 80 + x / 11
 7,8 x 11 = 80 + x
85,8 - 80 = x
  5,8 = x
jadi nilai dodi = 5,8
