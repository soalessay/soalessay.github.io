37. Sebutkan format penyimpanan pada aplikasi pengolah kata pengolah lembar kerja dan pengolah bahan presentasi​


Jawaban:

Format penyimpanan pada aplikasi pengolah kata, pengolah lembar kerja, dan pengolah bahan presentasi umumnya dapat berikut format-file umum yang digunakan dalam aplikasi ini:

1. Pengolah Kata (Word Processor):

- .doc/.docx: Format file standar yang digunakan dalam Microsoft Word.

- .rtf: Rich Text FormatFormat-file untuk pengolah lembar kerja (Spreadsheet):

- .xls/.xlsx: Format file standar yang digunakan dalam Microsoft Excel.

- .ods: OpenDocument Spreadsheet, format file terbuka yang digunakan oleh aplikasi seperti LibreOffice Calc.

Format-file untuk pengolah bahan presentasi (Presentation):

- .ppt/.pptx: Format file standar yang digunakan dalam Microsoft PowerPoint.

- .odp: OpenDocument Presentation, format file terbuka yang digunakan oleh aplikasi seperti LibreOffice Impress.

Selain format-file di atas, ada juga format-file lain yang dapat digunakan dalam masing-masing aplikasi pengolah kata, pengolah lembar kerja, dan pengolah bahan presentasi, tergantung dari aplikasi yang digunakan dan kebutuhan pengguna.
