14. Soal crita:Presiden Deborah kembali ke Floptropica setelah Perang Badussy dan serangan rudal Da Boyz minggu lalu.Presiden Deborah menaiki pesawat Mathilda Airlines yang mampu membawa 90 penumpang dan 2.280 kg bagasi.Penumpang dibagi menjadi 2 kelas, yaitu kelas ekonomi dan kelas bisnis.Setiap penumpang kelas ekonomi dapat membawa tidak lebih dari 20 kg bagasi dan penumpang kelas bisnis dapat membawa tidak lebih dari 30 kg bagasi.Harga tiket ekonomi Rp300.000,00 dan kelas bisnis Rp400.000,00. Jumlah penumpang kelas bisnis jika pendapatan pesawat dari penjualan tiket maksimum adalah . . . .HELP ! !​


Jawab:

Kapasitas pesawat Mathilda Airlines adalah 90 penumpang. Mari kita asumsikan x adalah jumlah penumpang kelas bisnis, maka jumlah penumpang kelas ekonomi adalah 90 - x.

Batasan berat bagasi untuk kelas ekonomi adalah 20 kg per penumpang, sehingga total berat maksimum bagasi yang dapat dibawa oleh penumpang kelas ekonomi adalah (90 - x) x 20 kg.

Batasan berat bagasi untuk kelas bisnis adalah 30 kg per penumpang, sehingga total berat maksimum bagasi yang dapat dibawa oleh penumpang kelas bisnis adalah x x 30 kg.

Total berat maksimum bagasi yang dapat diangkut oleh pesawat adalah 2.280 kg, sehingga kita dapat menulis persamaan:

(90 - x) x 20 + x x 30 = 2.280

Simplifikasi:

1.800 - 20x + 30x = 2.280

10x = 480

x = 48

Jadi, jumlah penumpang kelas bisnis adalah 48, dan jumlah penumpang kelas ekonomi adalah 90 - 48 = 42.

Pendapatan maksimum dari penjualan tiket dapat dihitung sebagai berikut:

Pendapatan dari kelas ekonomi = 42 x Rp300.000,00 = Rp12.600.000,00

Pendapatan dari kelas bisnis = 48 x Rp400.000,00 = Rp19.200.000,00

Jadi, pendapatan pesawat dari penjualan tiket maksimum adalah Rp31.800.000,00.

kelas bisnis = y

x, y ≥ 0

2x + 3y = 1000 ║x1║ 2x + 3y = 1000

Model matematikanya adalah x + y ≤ 48; 2x + y ≤ 75; f(x,y) = (Rp.1.200.000,00)x + (Rp.500.000,00)y

Pembahasan

Model matematika adalah suatu metode menyederhanakan suatu permasalahan dalam kehidupan sehari-hari agar mempermudahnya dalam perhitungan.

Penyelesaian

Diketahui:

1. Tempat duduk maksimum 48 orang

2. Bagasi:

Bisnis -> 40 kg/orang

Ekonomi -> 20 kg/orang

Total maksimum -> 1500 kg

3. Harga Tiket:

Bisnis -> Rp.1.200.000,00/orang

Ekonomi -> Rp.500.000,00/orang

Misal:

x -> Banyak penumpang Bisnis

y -> Banyak Penumpang Ekonomi

Ditanyakan:

Model Matematika

Jawab:

1. Tempat duduk maksimum 48 orang.

Karena pernyataan tersebut mengatakan jumlah orang yang ada dalam pesawat maka:

Penumpang Bisnis + Penumpang ekonomi

=> x + y

Karena maksimum 48 orang, maka jumlah penumpang bisa 48 orang tapi tidak mungkin lebih dari 48 orang.

=> x + y ≤ 48

2. Kapasitas bagasi.

Diketahui kelas bisnis kapasitasnya 40 kg per orang dan kelas ekonomi 20 kg per orang.

=> x(40 kg) + y(20 kg)

Total maksimum kapasitas bagasi 1500 kg. Sehingga tidak mungkin lebih dari 1500 kg.

=> x(40 kg) + y(20 kg) ≤ 1500 kg

Sederhanakan bentuknya dengan membagi tiap bilangan dengan FPB-nya yaitu 20.

=> 2x + y ≤ 75

3. Harga tiket.

Harga tiket kelas bisnis Rp.1.200.000,00 dan kelas ekonomi Rp.500.000,00. Sehingga total pendapatannya sesuai dengan banyaknya penumpang.

=> f(x,y) = (Rp.1.200.000,00)x + (Rp.500.000,00)y

Penjelasan dengan langkah-langkah:

Soal di atas dapat diselesaikan dengan membuat model matematikanya terlebih dahulu. Langkah untuk mengubah soal cerita menjadi model matematika adalah dengan menuliskan semua hal penting dari soal, menentukan variabelnya, membuat tabel dan membuat model matematikanya. Tabel untuk membuat model matematika pada soal diatas dapat dilihat pada lampiran.

Jumlah penumpang dari pesawat tersebut adalah 410 orang. Setiap penumpang ekonomi boleh membawa bagasi 20 kg. Sedangkan bisnis boleh membawa bagasi 30 kg. Pesawat dapat mengangkut bagasi paling banyak 8.800 kg.

Model matematika untuk soal diatas adalah:

x + y = 410

20x + 30y = 8.800 disederhanakan menjadi 2x + 3y = 880

x , y ≥ 0

Setelah dibuat model matematikanya kemudian dibuat grafik dari model matematika tersebut dan dicari titik potong grafik tersebut dengan sumbu x dan sumbu y. Untuk persamaan pertama titik potong dengan sumbu x dan sumbu y adalah:

x + y = 410 → x = 0 → 0 + y = 410 → y = 410

Titik potong = ( 0 , 410 )

x + y = 410 → y = 0 → x + 0 = 410 → x = 410

Titik potong = ( 410 , 0 )

Untuk persamaan kedua titik potong dengan sumbu x dan sumbu y adalah:

2x + 3y = 880 → x = 0 → 0 + 3y = 880 → y = 293,33

Titik potong = ( 0 , 293,33 )

2x + 3y = 880 → y = 0 → 2x + 0 = 880 → x = 440

Titik potong = ( 440 , 0 )

Daerah penyelesaian dari soal diatas adalah daerah antara titik ( 0 , 0 ), ( 410 , 0 ), ( 0 , 293,33 ) dan titik potong kedua garis. Untuk mencari titik potong kedua garis digunakan cara eliminasi dan substitusi dari nilai x dan y dari kedua persamaan diatas. Langkah pertama eliminasi variabel x dari kedua persamaan di atas.

2x + 3y = 880     | x1

x + y = 410          | x2

2x + 3y = 880

2x + 2y = 820  -

y = 60

Substitusi nilai y ke persamaan 1

x + 60 = 410

x = 410 - 60

x = 350

Jadi titik potong dari kedua garis tersebut adalah ( 350 , 60 ). Setelah itu dilakukan pengujian dari ketiga titik potong tersebut dengan memasukkan nilai keuntungan dari setiap kolam

( 410 , 0 ) → 410 x 400.000 → 164.000.000

( 0 , 293.33 ) → 293,33 x 600.000 → 175.998.000

( 350 , 60 ) → 350 x 400.000 + 60 x 600.000 → 140.000.000 + 36.000.000 → 176.000.000

Pendapatan maksimum ada pada titik potong kedua garis persamaan yaitu pada titik ( 350 , 60 ). Pendapatan maksimum yang bisa didapatkan perusahaan penerbangan tersebut adalah Rp 176.000.000.
