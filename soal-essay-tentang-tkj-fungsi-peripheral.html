Peripheral utama
contohnya: mouse, keyboard dan monitor.

Peripheral Pendukung.
contohnya: printer, scaner, modem, dan lain-lain.

Jawaban dan Penjelasan :Jenis-Jenis Perangkat Peripheral

Ada banyak perangkat peripheral yang berbeda, tetapi dapat dibagi ke dalam tiga kategori umum sebagai berikut:

1. Perangkat Input (Input Device), seperti Keyboard dan Mouse

Perangkat input adalah bagian dari perangkat keras komputer yang digunakan untuk menyediakan data dan sinyal kendali ke komputer.

2. Perangkat Output (Output Device), seperti Monitor dan Printer

Perangkat output adalah bagian dari perangkat keras komputer yang menggunakan data dan perintah dari komputer Anda untuk melakukan suatu tugas.

Dengan perangkat output, hasil pengolahan data yang dilakukan oleh sistem pengolah informasi seperti komputer yang mengubah informasi yang dihasilkan secara elektronik menjadi bentuk yang dapat dibaca manusia.

3. Perangkat Penyimpanan (Storage Device), seperti Hard Disk Drive dan Cloud Storage

Perangkat penyimpanan adalah perangkat keras komputasi yang digunakan untuk meany implanted, memetakan dan mengekstrak file data dan objek.

Perangkat penyimpanan dapat menyimpan informasi baik untuk sementara maupun permanen, dan dapat bersifat internal atau eksternal ke komputer, server atau perangkat komputasi sejenisnya.

33. Sebutkan macam macam peripheral menurut fungsi dan kegunaan!​


Jawaban:

Berikut adalah beberapa contoh peripheral atau perangkat keras komputer yang bisa dibagi berdasarkan fungsi dan kegunaannya:

1. Input Device (perangkat masukan):

- Keyboard: untuk memasukkan data atau karakter ke dalam sistem komputer

- Mouse: untuk mengontrol cursor atau pointer pada layar dan memudahkan navigasi atau pemilihan objek

- Scanner: untuk mengubah dokumen atau gambar menjadi data digital pada komputer

- Mikrofon: untuk merekam suara dan memasukkannya ke dalam sistem

- Kamera: untuk merekam gambar dan video dan memasukkannya ke dalam sistem

2. Output Device (perangkat keluaran):

- Monitor: untuk menampilkan tampilan atau output sistem komputer

- Printer: untuk mencetak dokumen atau gambar

- Speaker: untuk menjalankan atau memutarkan suara atau musik

- Proyektor: untuk memproyeksikan layar atau output sistem pada ukuran yang lebih besar

3. Storage Devices (perangkat penyimpanan)

- Hard drive: untuk menyimpan data dalam jangka panjang pada komputer

- Solid-state drive (SSD): penyimpanan data tanpa menggunakan mekanik pada hard drive

- USB drive: menyimpan data pada eksternal drive dan dapat dibawa kemana saja

- Memory card: menggunakan teknologi flash memory untuk menyimpan data seperti foto dan video pada kamera atau ponsel

4. Network Devices (perangkat jaringan)

- Router: meneruskan data dalam jaringan, memberikan pengaturan keamanan, dan menghubungkan beberapa perangkat

- Switch: melakukan pengiriman paket data di jaringan dan menghubungkan beberapa perangkat

- Modem: mengubah sinyal digital menjadi sinyal analog untuk dapat dihubungkan ke internet melalui jaringan telepon atau kabel

- Network Interface Card (NIC): menghubungkan komputer atau perangkat ke jaringan melalui kabel atau nirkabel.

40. sebutkan soal essay komposisi 3 fungsi


Diberikan dua buah fungsi masing-masing f(x) dan g(x) berturut-turut adalah:

f(x) = 3x + 2
g(x) = 2 − x

Tentukan:
a) (f o g)(x)
b) (g o f)(x)

Pembahasan
Data:
f(x) = 3x + 2
g(x) = 2 − x

a) (f o g)(x)

"Masukkan g(x) nya ke f(x)"

sehingga:
(f o g)(x) = f ( g(x) )
= f (2 − x)
= 3(2 − x) + 2
= 6 − 3x + 2
= − 3x + 8

b) (g o f)(x)

"Masukkan f (x) nya ke g (x)"

sehingga:
(g o f)(x) = g ( f (x) )
= g ( 3x + 2)
= 2 − ( 3x + 2)
= 2 − 3x − 2
= − 3x
