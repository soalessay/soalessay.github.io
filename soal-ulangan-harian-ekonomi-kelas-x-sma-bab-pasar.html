Frekuensi relatif ( frekuensi nisbi/peluang empiris )
frekuensi relatif muncul A = [tex] \frac{sering muncul A}{banyak lemparan} [/tex]
frekuensi relatif muncul G = [tex] \frac{sering muncul G}{banyak lemparan} [/tex]
Contoh :
Mata uang dilemapar = 80 kali
muncul A = 35 kali
muncul G = 45 kali
f.muncul A = [tex] \frac{35}{80} = \frac{7}{16} [/tex]
f. muncul G = [tex] \frac{45}{80} = \frac{9}{16} [/tex]

Ruang sampel dan titik sampel
a. 1 mata uang dilemparkan
S = { A, G }
n ( s ) = 2
b. 1 dadu dilemparkan
S = { 1, 2, 3, 4, 5, 6 }
n ( S ) = 6
c. 2 mata uang yg dilemparkan
uang1       A             G

( gamabr diagram pohonnya )

Kisaran Peluang
peluang = kemungkinan/probability/ probabilitas
P = [tex] \frac{jumlah yg dimaksud}{jumalah yg mungkin} [/tex]
P ( E ) = [tex] \frac{n ( E ) }{n ( S )} [/tex]
contoh 
3 mata uang dilempar bersamaan
P ( 3A ) = [tex] \frac{1}{8} [/tex]
P ( A ) + P ( A' ) = 1
peluang hujan + peluang tdk hujan = 1
Frekuensi Harapn
= P × banyaknya jawaban
contoh :
P. siswa kelas X A tidak naik kelas = [tex] \frac{1}{6} [/tex]

Berikut skedul permintaan (jumlah permintaan pada berbagai harga) dengan mengambil range harga barang X dari 0 sampai 6 dollar (asumsi satuan P adalah dollar / $):

[tex]\boxed{\begin{array}{c|c} \sf Q_d &\sf P~(\$)\\\sf \_\_\_\_\_\_ &\sf \_\_\_\_\_\_\_\_\\\sf 0 &\sf 6\\\sf 5 &\sf 5\\\sf 10 &\sf 4\\\sf 15 &\sf 3\\\sf 20 &\sf 2\\\sf 25 &\sf 1\\\sf 30 &\sf 0\\ \end{array}}[/tex]

Tabel ditulis dengan mengambil asumsi bahwa harga barang dalam satuan Rupiah. Adapun kurva permintaan barang dari skedul permintaan di atas dapat dilihat pada lampiran.

Penjelasan dengan langkah-langkah

Skedul permintaan dapat didefinisikan sebagai alternatif nilai dari jumlah barang yang diminta untuk setiap nilai harga tertentu. Rentang nilai dan harga ini diambil asumsi saja mengingat pada soal tidak diberikan, seperti diambil rentang atau range jumlah barang yang diminta (Qd) dari 0 sampai 8 buah barang saja.

Diketahui:

Fungsi permintaan:
Qd = 30 - 5PKeterangan:
Qd adalah jumlah barang X yang diminta pembeli.
P adalah nilai harga dari barang X yang diminta.Asumsi ceteris paribus (hanya P yang mempengaruhi Qd dan sebaliknya, atau variabel lain dianggap tetap).Diambil P:
0, 1, 2, 3, 4, 5, dan 6.

Ditanyakan:

Skedul permintaan = ?Kurva permintaan = ?

Penyelesaian:

Langkah 1
Perhitungan nilai Qd dari setiap nilai P.

Untuk P = 0
Qd = 30 - 5P
Qd = 30 - 5(0)
Qd = 30.Untuk P = 1
Qd = 30 - 5P
Qd = 30 - 5(1)
Qd = 25.Untuk P = 2
Qd = 30 - 5P
Qd = 30 - 5(2)
Qd = 20.Untuk P = 3
Qd = 30 - 5P
Qd = 30 - 5(3)
Qd = 15.Untuk P = 4
Qd = 30 - 5P
Qd = 30 - 5(4)
Qd = 10.Untuk P = 5
Qd = 30 - 5P
Qd = 30 - 5(5)
Qd = 5Untuk P = 6
Qd = 30 - 5P
Qd = 30 - 5(6)
Qd = 0.

Langkah 2
Menggambar kurva permintaan.
