35. Carilah informasi tentang pemrograman web​


Jawaban: Pemrograman Web

Penjelasan:

adalah proses atau cara untuk menjalankan intruks Pada sebuah komputer yang terhubung ke internet untuk membuat fungsi atau tugas tertentu

Jawaban dan Penjelasan:

Pemrograman web, atau web programming, adalah aktivitas menulis kode-kode program dalam proses atau tahapan pengembangan/pembangunan web (web development), yang di antaranya meliputi konten web, scripting web dari sisi client dan server, serta keamanan jaringan, terkait dengan web yang dibangun.

Pemrograman web berbeda dengan hanya “memprogram” atau “membuat program”, karena memerlukan pengetahuan interdisipliner (antara beberapa disiplin ilmu yang berhubungan) pada area aplikasi, scripting di sisi client dan server, serta teknologi basisdata.

Oleh karena itu, secara garis besar, terdapat dua buah kategorisasi pemrograman web, yaitu:

Pemrograman web pada sisi client
Pada sisi client, pemrograman diperlukan dalam hal pengaksesan data dan penyajian informasi dari dan kepada pengguna web.
Pemrograman web pada sisi client juga diperlukan untuk meningkatkan pengalaman pengguna (user experience) dalam berinteraksi dengan web yang dikunjungi.
