Soal ini berkaitan dengan konsep ekonomi makro dan perhitungan GDP (Gross Domestic Product) dengan menggunakan pendekatan pengeluaran (expenditure approach).

a. Untuk menghitung nilai GDP, kita perlu menggunakan persamaan pengeluaran agregat:

GDP = C + I + G + NX

dimana:

C = Konsumsi

I = Investasi

G = Belanja pemerintah

NX = Ekspor - Impor

Dalam soal ini, kita telah diberikan nilai I dan G. Kita perlu mencari nilai C dan NX terlebih dahulu sebelum dapat menghitung GDP.

Untuk mencari nilai C, kita dapat menggunakan persamaan konsumsi:

C = 40 + 0.9 YD

YD adalah pendapatan disposabel (disposable income), yang dapat dihitung dengan cara:

YD = Y - T

dimana:

Y = Pendapatan nasional

T = Pajak

Dalam soal ini, kita telah diberikan nilai T. Kita perlu mencari nilai Y terlebih dahulu sebelum dapat menghitung YD dan C.

Untuk mencari nilai Y, kita dapat menggunakan persamaan pendapatan:

Y = C + I + G

Kita sudah memiliki nilai I dan G. Kita dapat mengganti nilai C dengan persamaan konsumsi:

Y = (40 + 0.9 YD) + I + G

Kita juga telah diberikan nilai I dan G. Kita dapat mengganti nilai YD dengan persamaan YD = Y - T:

Y = (40 + 0.9(Y - 200)) + 100 + 250

Setelah melakukan perhitungan, kita dapatkan nilai Y sebesar $ 1950 milyar. Selanjutnya, kita dapat menghitung nilai YD dan C:

YD = Y - T = 1950 - 200 = $ 1750 milyar

C = 40 + 0.9 YD = 40 + 0.9(1750) = $ 1615 milyar

Terakhir, kita dapat menghitung GDP:

GDP = C + I + G + NX

GDP = $ 1615 milyar + $ 100 milyar + $ 250 milyar + NX

Jadi, nilai GDP pada tahun 2017 sebesar $ 1965 milyar.
