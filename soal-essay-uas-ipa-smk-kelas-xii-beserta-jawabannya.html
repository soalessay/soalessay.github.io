1. Tiga buah resistor akan dirangkai secara seri, masing-masing nilai resistor tersebut adalah R1 = 10Ω, R2 = 47Ω, R3 = 100Ω. berapakah nilai dari hambatan pengganti rangkaian tersebut?

Diketahui : R1 = 10Ω
R2 = 47Ω
R3 = 100Ω

Ditanya : Rs = ?

Jawab :

Rs = R1 + R2 + R3 

Rs = 10Ω + 47Ω + 100Ω = 157 Ω
