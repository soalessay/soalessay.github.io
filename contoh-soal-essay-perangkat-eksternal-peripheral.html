1. pengertian perangkat eksternal peripheral


Adalah perangkat yang digunakan untuk memasukkan data atau perintah ke dalam komputer

2. pengertian perangkat eksternal peripheral


PERANGKAT EKSTERNAL
Mesin komputer akan memiliki nilai apabila bisa berinteraksi dengan dunia luar. Lebih dari itu, komputer tidak akan berfungsi apabila tidak dapat berinteraksi dengan dunia luar. Ambil contoh saja, bagaimana kita bisa menginstruksikan CPU untuk melakukan suatu operasi apabila tidak ada keyboard. Bagaimana kita melihat hasil kerja sistem komputer bila tidak adamonitor.Keyboard dan monitor tergolong dalam perangkat eksternal komputer.Perangkat eksternal yang dihubungkan modul I/O seringkali disebut perangkat peripheral, atau untuk mudahnya disebut peripheral.Sistem komputer tidak akan berguna tanpa adanya peralatan input dan output. Operasi-operasi I/O diperoleh melalui sejumlah perangkat eksternal yang menyediakan alat untuk pertukaran data di antara lingkungan luar dengan komputer. Perangkat eksternal dihubungkan dengan komputer oleh suatu link dengan modul I/OLink digunakan untuk pertukaran kontrol, status, dan data antara modul I/O sering kali disebut sebagai perangkat peripheral, atau untuk mudahnya disebut peripheral.

yang dimaksud peripheral adalah perangkat tambahan yang dimasukkan / ditambahkan kedalam komputer. perangkat ini digunakan sebagai tambahan atau bisa saja sebagai perangkat inti pembangun komputer (bukan termasuk CPU)

contoh perangkatnya :
1. input devices = perangkat yang berfungai untuk memasukkan data kedalam komputer.
contoh : keyboard, scanner, dll.

2. output devices = perangkat yang berfungsi untuk mengeluarkan hasil dari proses komputer.
contoh : monitor, printer, speaker, dll.

3. storage devices = perangkat yang berfungsi untuk menyimpan data dikomputer secara digital. perangkat penyimpanan yang wujudnya nyata seperti hardisk, SSD, flashdisk, dll.
