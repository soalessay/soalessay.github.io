Program Mengitung Luas Persegi Panjang Dalam PASCAL

luas Persegi Panjang

Program menghitung luas persegi panjang. Pada Kesempatan kali ini saya akan membagi sebuah tutorial cara membuat program menghitung luas persegi panjang. Sebelum membuat programnya kita perlu mendefiniskan variabel yang akan digunakan seperti : p (Panjang), l (Lebar), dan lu (luas). setelah itu ita buat algoritmanya atau dalam artian logiakanya.

-* Mulai

-* Masukkan Nilai Panjang

-* Masukkan Nilai Lebar

-* Luas sama dengan panjang kali lebar

-* Cetak Lusa

-* Selesai.

setelah meliahat algoritma diatas kita sudah bisa membuat programnya.

Program Luas_Ppanjang;

uses crt;

Var l,p : integer

lu : real

begin

clrscr;

write(‘Masukkan Nilai Panjang : ‘);readln(p);

write(‘Masukkan Nilai Lebar : ‘);readln(l);

lu := p*l;

writeln(‘Nilai Luas Persegi Panjang= ‘,lu);

readln;

end.

Program Luas_Ppanjang; Sebuah judul program boleh digunakan boleh tidak, kalau ingin digunakan harus sesuai prosedur dengan program nama judul;

Uses crt; Sebuah defenisi program boleh ditulis boleh tidak. Biasanya apabila ingin menggunakan menu window clrscr; harus menggunakan uses crt;

Var Sebuah deklarasi variabel, dimana setiap variabel di berikan type datnya dan bentuk datanya dapat berubah contoh : p, l : integer; berarti nilai p dan l adalah bilangan bulat

Begin akhiri dengan end. Merupakan blok program yang mengandung pernyataan – pernyataan dan intruksi – intruksi baris program yang dibuat.

Clrscr; Untuk membersihkan layar dari program sebelumnya. Setiap penggunaannya harus didefinisikan dengan uses crt;

Write(‘Masukan Nilai panjang : ‘); adalah untuk menampilkan inis program dimana cursor berikutnya akan berada di smaping tulisan yang dibuat. Contoh diatas diminta untuk mencetak Masukkan Nilai panjang :_.

Readln ( ); Untuk membaca suatu variabel atau program

lu := p*l; sebuah proses pemberian niali suatu variabel. Dimana Variabel lu adalah jumlah dari hasil kali variabel p dan l.

Readln; untuk melihat isi program setelah program di jalankan

Cara Deklarasi Array

Array dideklrasikan di bagain deklrasi sama seperti variabel lainya dengan menggunakan kata kunci ARRAY dan tipe data yang anda tentukan serta batas indeks array yang diapit oleh bracket ( [ ] ) seperti contoh di bawah ini :namaArray : array [ indeks awal .. indeks akhir ] of tipe data ;

Contoh Penulisan array A dengan tipe data integer :

A : array [ 1.. 10 ] of integer ;

Contoh Penulisan array A dengan tipe data string, real , char :

A : array [ 1.. 10 ] of string ;

B : array [ 1..5 ] of real ;

D : array [ 1..15 ] of char

Perbedaan antara array 1 dimensi dan array 2 dimensi

Array 1 dimensi terdiri dari satu baris dan banyak kolom, Sedangkan array 2 dimensi terdiri dari banyak baris dan banyak kolom

Cara Mendeklarasikan Array 2 Dimensi / Matrik

Var namaarray : array [ 1..5,1..10] of integer ;

Type

28. Penjelasan Hukum Pascal dengan Contoh Soalnya​


Mapel : Fisika

Materi : Hukum Pascal

Kata kunci : Hukum pascal

⭐

Hukum Pascal (baca: [paskal]) menyatakan bahwa Tekanan yang diberikan zat cair dalam ruang tertutup diteruskan ke segala arah dengan sama besar[1].

Perbedaan tekanan karena perbedaan kenaikan zat cair diformulakan sebagai berikut:

ΔP=pg(Δh)

ΔP adalah tekanan hidrostatik (dalam satuan pascal atau "Pa"), atau perbedaan tekanan pada 2 titik dalam sekat yang berisi zat cair, karena perbedaan berat antara keduanya;

ρ adalah massa jenis zat cair (dalam kilogram per meter kubik);

g adalah percepatan karena gravitasi (umumnya menggunakan percepatan ketinggian dari permukaan laut akibat gravitasi bumi, dalam satuan meter per detik pangkat 2);

Δh adalah ketinggian zat cair di atas titik pengukuran (dalam satuan meter), atau perbedaan ketinggian antara 2 titik pada kolom yang berisi zat cair .

40. contoh soal hukum Pascal 1?​


Jawaban:

Sebuah tabung bertekanan dengan luas penampang 0,02 m2 dihubungkan dengan pipa sempit dengan luas penampang 0,001 m2. Jika sebuah piston ditekan ke bawah dengan gaya 500 N, berapa gaya yang akan diterima oleh fluida yang ada di dalam tabung? Anggap fluida dalam tabung adalah zat tidak dapat dipadatkan.

Diketahui:

A1 = 0,02 m2

A2 = 0,001 m2

F1 = 500 N

F2 = ?

Ditanya:

Besarnya F2

Jawaban:

Hukum Pascal 1 menyatakan bahwa tekanan hidrostatis di dalam fluida yang tertutup dan tidak dapat dipadatkan akan merata di seluruh bagian fluida, sehingga besarnya tekanan di suatu titik di dalam fluida pada kedalaman yang sama akan selalu sama.

Dari hukum ini, kita dapat menggunakan persamaan:

F1 / A1 = F2 / A2

Substitusikan nilainya:

500 N / 0,02 m2 = F2 / 0,001 m2

F2 = 500 N / 0,02 m2 x 0,001 m2

F2 = 25 N

Jadi, gaya yang diterima oleh fluida yang ada di dalam tabung adalah sebesar 25 N.
